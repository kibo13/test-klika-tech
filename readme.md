## Quick Start

```bash
# download or clone repository
git clone git@github.com:kibo13/test-klika-tech.git

# import to created database file 
static/db_music.sql 

# server side 
cd server/ 

# make settings in .env file 

# install dependencies
npm install

# run script 
npm start 

# client side  
cd client/ 

# install dependencies
npm install

# run script 
npm start 
```
